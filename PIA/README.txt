==========================
  Spotify Searcher App
==========================
Bienvenido a nuestro proyecto Spotify Searcher App!!
En este software se dan diferentes opciones entre ellas: Buscar discografías de un artista, Consular reportes, Consultar grafías, entre otros.
Esto funciona haciendo una búsqueda de un artista o una banda, y da diferentes datos como los álbumes con mas canciones, la duración de cada álbum y la fecha de publicación. Se pueden consultar los registros y graficas.

Este proyecto fue hecho con Python y el API de Spotify.

Este proyecto ocupa la instalación de las siguientes librerías:
openpyxl
matplotlib
pillow
Para instalar las librerías se ejecuta el archivo "librerias.bat" o bien se ingresa al cmd y poner el comando: "pip install" seguido por la librería
